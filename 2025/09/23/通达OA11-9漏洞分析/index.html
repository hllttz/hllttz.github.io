组合链

我们已经知道/mobile/api/api.ali.php可以上传 json 文件，而inc\package\work.php可以实现读取 json 文件，可以组合起来形成一条代码执行的利用链。但还有一个问题需要解决，就是上传的文件名格式中存在一个随机数,无法猜测,也不能用?通配符

但可以利用 windows 上的一个特性

windows系统在处理文件名的时候，FindFirstFileExW()/FindFirstFile()这两个API会对< > "这三个字符做特殊处理，效果分别对应Linux下的通配符* ? .

所以我们可以用>来代替?实现匹配文件，如此便可成功构造出一条前台 getshell 利用链。最后使用 python 编写一键 getshell 脚本
